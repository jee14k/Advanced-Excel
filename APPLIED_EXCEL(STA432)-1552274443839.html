Course Objectives/Course Description:

To build the logical thinking ability and to provide hands-on experience in solving statistical models using MS Excel with Problem based learning. To explore and visualize data using excel formulas and filters. To learn working with different kinds of data into excel

Learning Outcome

After completing this course the students will be able to,

1. Clearly understanding of logics of using excel features

2. Analyze the given problem and solve using Excel

3. Building blocks of excel, excel shortcuts, sample data creation and analyzing data

Unit-1Teaching Hours:2

Basics

Introduction: File types, Spreadsheet structure, Menu bar, Quick access toolbar, Mini toolbar, Excel options. Formatting: Format painter, Font, Alignment, Number, Styles, Cells, Clear, Page layout.

Unit-2Teaching Hours:8

Data collection

Overview of Excel tables and properties, Collecting sample data and arranging in definite format in excel tables, data validation, using form for data entry, Secondary data collection from different databases. (Different types of data should be collected to use in next chapters)

Unit-3Teaching Hours:6

Importing and exporting data

Importing data from different sources - text file, web page and XML file, Exporting data in different formats - text, csv, image, pdf etc, Creating database with the imported data.

Unit-4Teaching Hours:8

Data cleaning and naming range

Data tools text to column, identifying and removing duplicates, using format cell options, Application of functions - Concatenate, upper, lower, year, month, day, text, year, Examples of using named ranges in SUM, AVERAGE

Unit-5Teaching Hours:6

Handling missing data

Data manipulation in table using shortcuts, using formulas and function, Missing value handling in Graph using example of scatter graph with connecting line.

Unit-6Teaching Hours:8

Excel functions and formulas

LOOKUP, VLOOKUP, HLOOKUP, INDEX, MATCH

Unit-7Teaching Hours:10

Data analysis

Getting started with data analysis toolpak, descriptive statistics (using basic functions and data analysis toolpak), correlation, linear regression, one way ANOVA (with interpretations).

Unit-8Teaching Hours:12

Macros and Security

Introduction to macros, using macros for data entry, importing files, Data cleaning and managing using macro, Different types of security available in Excel to protect the contents.
